# Rule-Based Chatbot

## Overview
This project is a simple Rule-Based Chatbot developed using Python. It responds to predefined user inputs using if-elif-else statements.

## Features
- Greeting responses
- Help command
- Exit command
- Continuous conversation
- Rule-based decision making

## Technologies Used
- Python

## How to Run

```bash
py chatbot.py
```

## Sample Commands

- hello
- hi
- how are you
- your name
- thanks
- help
- exit
